# LOJA DE ROUPA - ESTOQUE

Sistema de loja de roupas desenvolvido em Java, utilizando JDBC e MySQL para controle de estoque.

## Funcionalidades
- Cadastro de produtos
- Consulta por marca, tamanho, cor e modelo
- Controle de estoque
- Integração com banco de dados MySQL

## Tecnologias
- Java
- JDBC
- MySQL
- IntelliJ IDEA
